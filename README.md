# telepot - Python framework for Telegram Bot API

**[Bot API 3.0](https://core.telegram.org/bots/api)** has come out, the biggest
new feature being **[Payments](https://core.telegram.org/bots/payments)**.
Barring unexpected obstacles, you can expect an update to telepot on or before
**May 31, 2017**, encompassing all those new developments.
